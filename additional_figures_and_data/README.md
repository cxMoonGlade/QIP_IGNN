# Additional figures and data (paper supplement)

This folder holds material that may be **omitted from a short conference PDF** for space, while keeping it **versioned with the code** that reproduces the experiments.

Contents:

| File | Description |
|------|-------------|
| `bp_main_3x2.pdf` | Barren-plateau-style **gradient-variance diagnostic** figure (former Fig.~3 in the paper draft). |
| `gradient_landscape_diagnostic_from_paper.tex` | LaTeX snippet copied from the paper source: **Gradient-Landscape Diagnostic** subsection and figure environment. `\includegraphics` points at `bp_main_3x2.pdf` in this directory. |
| `broader_generalization_from_paper.tex` | LaTeX snippet copied from the paper source: **Broader Generalization Diagnostics** appendix section and table. |

**Note:** These snippets assume a LaTeX preamble that already loads `graphicx`, `booktabs`, etc., as in the paper. They are meant as a stable archive of the exact wording used in the full draft, not as standalone compilable documents without edits.

Repository root: [QIP_IGNN](https://github.com/cxMoonGlade/QIP_IGNN).
